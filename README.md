# Sistema de Arquivos em C

Projeto desenvolvido em linguagem C para simular um sistema de arquivos hierárquico semelhante aos utilizados em sistemas Unix/Linux.

## Funcionalidades Implementadas

* Criação de diretórios (mkdir)
* Listagem de diretórios (ls)
* Estrutura baseada em árvores
* Alocação dinâmica de memória

## Estruturas de Dados Utilizadas

* Árvores
* Listas encadeadas
* Ponteiros

## Como Compilar

```bash
gcc main.c directory.c -o sistema.exe
```

## Como Executar

```bash
./sistema.exe
```
